
Un café chaud, 
le ciel est clair.
J'ai des idées 
et du temps à faire.

Je peux le faire,
J'avance c'est tôt 
même si parfois, je tombe un peu.
Un mot, un pas, un jour de plus, 
je peux le faire, je vais jusqu'au bout.

Je cherche les mots, 
je tends l'oreille.
Je fais des pauses 
quand le cœur sommeille.

Je peux le faire,
J'avance tôt 
même si parfois, je doute un peu.
Un mot, un pas, un jour de plus.
Je peux le faire, je vais jusqu'au bout.

Parfois je râle, 
parfois j'oublie.
Mais chaque erreur 
me montre la vie.

Je peux le faire.
J'avance c'est tôt 
même si parfois, je perds le goût.
Un mot, un pas, un jour de plus,
je peux le faire, je vais jusqu'au bout.

Je lève les yeux, 
je vois plus loin.
Ce que je veux 
n'est pas si loin.

Je peux le faire.
J'avance tôt, 
même si parfois le ciel est flou.
Un mot, un pas, un jour de plus, 
je peux le faire, je vais jusqu'au bout.
